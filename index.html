⚙ Fuentes de datos

Orden de prioridad solicitado: SofaScore → FotMob → Flashscore → football-data.org → APIs públicas de fútbol. En la práctica, SofaScore, FotMob y Flashscore no se pueden consultar desde un sitio estático en GitHub Pages: no publican una API pública para frontend y bloquean las peticiones directas del navegador por CORS (y consultarlas mediante scraping incumpliría sus términos de servicio). Por eso esta app usa las dos únicas fuentes públicas que sí aceptan peticiones desde el navegador con una clave gratuita:

DatoAPI real disponibleSin clave configurada
Calendario de partidos football-data.org (gratis) Calendario oficial de la fase de grupos, verificado manualmente y embebido en la app (no en vivo, pero no inventado)
Cuotas Bet365 / casas de apuestas The Odds API (gratis, capa limitada) Cuotas simuladas a partir de las probabilidades del modelo + margen de casa típico
xG, tiros, córners, tarjetas, posesión Ninguna API gratuita de frontend las ofrece (SofaScore/FotMob/Understat/FBref no tienen API pública) Siempre estimadas por el modelo propio a partir del nivel de cada selección
Lesiones, sanciones, alineaciones Ninguna API gratuita las ofrece Siempre estimadas de forma ilustrativa, nunca la convocatoria oficial

Todo dato estimado se marca en la interfaz con la etiqueta ESTIMADO. La app nunca presenta una cifra simulada como si fuera un dato confirmado.

Las claves se guardan únicamente en el localStorage de tu navegador. Nunca se envían a ningún servidor de terceros salvo a la propia API que elijas. Si una petición en vivo falla (red, CORS, límite de uso), la app no inventa datos: muestra un aviso y usa el calendario oficial embebido o las cuotas simuladas, dejándolo siempre indicado en pantalla.

MUNDIAL26 AI Analytics — Proyecto educativo de análisis estadístico. Las predicciones se generan con un modelo propio (distribución de Poisson + scoring ponderado) y no garantizan resultados. No constituye asesoramiento financiero ni recomendación de apuesta real.
