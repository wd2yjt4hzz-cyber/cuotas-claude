ℹ️ Fuentes de datos y modo demo

Integración preparada para API tipo SofaScore / The Odds API

Esta aplicación funciona 100% en frontend (HTML + CSS + JS) y está pensada para GitHub Pages. Por defecto usa un motor de simulación estadística calibrado con el nivel de cada selección. Todo dato simulado se marca visualmente como tal en la interfaz.

En el archivo script.js encontrarás una constante API_KEY = "TU_CLAVE_AQUI" y la estructura de fetch preparada para conectar con una API de deportes (estilo SofaScore). Sustituye esa clave por la tuya y ajusta la URL base al proveedor que elijas.

⚙ Fuentes de datos

Bet365, SofaScore, Flashscore, FotMob y FBref no ofrecen una API pública para frontend y bloquean las peticiones directas por CORS. Esta app usa por defecto un motor de simulación estadística calibrado con el nivel de cada selección (modo demo, totalmente funcional sin claves).
